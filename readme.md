## About This Repository

This repository contains the complete MATLAB source code, output figures, and documentation for all laboratory sessions of the Computational Engineering course. Each session is self-contained in its own folder and can be run independently without any additional MATLAB toolboxes.

**MATLAB R2020a or later is recommended. No extra toolboxes required.**

---


## Lab Sessions Summary

| Session | Topic | Key Methods | Problems Solved |
|---|---|---|---|
| Lab 01 | MATLAB Basics, Taylor Series & Numerical Errors | Maclaurin series, Euler method | sin(x) convergence, Falling parachutist, Quadratic equation |
| Lab 02 | Bisection Method | Bisection | Parachutist drag coefficient, sin10x + cos3x |
| Lab 03 | False Position Method | False Position | x¹⁰ − 1, Rocket velocity, Floating sphere |
| Lab 04 | Newton–Raphson & Fixed-Point Iteration | Newton–Raphson, FPI | Nonlinear spring system |

## Topics Covered

- Numerical errors: truncation error, round-off error, absolute and relative error
- Taylor and Maclaurin series convergence
- Bracketing methods: Bisection, False Position
- Open methods: Fixed-Point Iteration, Newton–Raphson, Secant, Modified Secant
- Hybrid algorithms: combining bracketing with open methods
- Euler's method for ODEs
- Engineering applications: plasticity models, pipe flow, structural dynamics
